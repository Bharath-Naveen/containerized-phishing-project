# Raw data (not committed)

This directory stays **empty in git**. Add files locally:

| Path | Purpose |
|------|---------|
| `kaggle/` | Drop the Kaggle CSV here (see `kaggle/README.txt` and `docs/DATASET_SETUP.md`). |
| `*.txt` | Optional challenge lists (`legit-*.txt`, `phish-*.txt`). Copy from `archive/sample_data/challenge/` to run challenge evaluation. |

Do **not** commit API keys, Kaggle JSON, or large downloaded datasets.
